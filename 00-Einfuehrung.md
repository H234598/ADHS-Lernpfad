# Einführung

## Ziel

Der Lernpfad baut zuerst begriffliches und klinisches, danach neuropsychologisches, neurobiologisches, diagnostisches und forschungsmethodisches Verständnis auf.

## Evidenzmarker

Jede zentrale Aussage kann mit zwei getrennten Markern versehen werden:

| Marker | Bedeutung |
|---|---|
| `evidence: consensus` | breit gestützter Konsens oder Leitlinie |
| `evidence: high` | mehrere hochwertige Reviews/Meta-Analysen |
| `evidence: moderate` | konsistente, aber begrenzte Evidenz |
| `evidence: preliminary` | frühe oder widersprüchliche Forschung |
| `status: consensus` | derzeitige Lehrmeinung |
| `status: probable` | plausibles Modell |
| `status: disputed` | umstritten |
| `status: experimental` | experimentell/offen |

Das **Studiendesign** allein bestimmt nicht automatisch die Qualität. Eine schlechte Meta-Analyse wird nicht durch ihr Etikett gut; eine sorgfältige Kohorte kann für bestimmte Fragen unverzichtbar sein.

## Gruppenbefunde

„Menschen mit ADHS zeigen …“ bedeutet gewöhnlich einen durchschnittlichen Gruppenunterschied. Es bedeutet nicht, dass jede betroffene Person dieses Merkmal besitzt oder dass ein Test daraus eine individuelle Diagnose ableiten kann.

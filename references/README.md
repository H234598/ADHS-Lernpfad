# Studienkarten

Jede Kernquelle besitzt eine eigene Markdown-Datei mit stabiler ID, Zitation, Evidenztyp, Kernaussagen und Einschränkungen. `Literatur.md` wird daraus automatisch erzeugt.

## Regeln

- Dateiname = stabile `reference_id`
- DOI ohne URL-Präfix speichern
- PubMed-ID separat speichern
- Kernaussagen paraphrasieren, nicht Abstracts kopieren
- methodische Grenzen sichtbar machen

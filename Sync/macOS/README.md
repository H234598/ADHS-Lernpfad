# macOS-Synchronisierung

Die Ordnerstruktur ist vorbereitet; ein produktives macOS-Paket wird später gemeinsam mit den übrigen noch offenen Plattformen geplant.

## Vorgesehene Varianten

- manuelles Shell-Skript
- regelmäßige Ausführung über den systemeigenen Dienstmechanismus
- Auswahl zwischen `safe-pull`, `prompt-pull`, `forced-pull` und einem späteren `full-sync`
- geschützte Obsidian- und Synchronisationsmetadaten

Die gemeinsame Modusdefinition steht unter [[Sync/MODES|Synchronisationsmodi]].

> [!info]
> Dieser Ordner enthält noch keine ausführbare Automatisierung. Die spätere Implementierung soll zuerst als Gesamtplan mit Tests, Deinstallation und Konfliktstrategie entworfen werden.
